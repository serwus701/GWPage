Конфигурация на IP адрес
Безжичното устройство трябва да бъде конфигурирано да използва IP адрес. Има два възможни варианта:

Автоматично получаване на IP адрес. Използвайте тази опция, ако мрежата ви използва DHCP сървър.

Задаване на статичен IP адрес. Използвайте тази опция, ако мрежата ви не използва DHCP сървър и желаете ръчно да въведете настройките за IP адрес и мрежова маска.

Ако не сте сигурни коя опция да изберете, моля говорете със системния администратор.

Моля, имайте предвид: ако изберете опцията “Автоматично получаване на IP адрес”, вашето устройство ще се опитва да получи IP адрес от DHCP сървър. Ако не успее да комуникира с DHCP сървъра, защото сървърът е недостъпен или защото не сте конфигурирали правилно настройките за безжична защита, устройството ще премине към IP адрес, базиран на APIPA в диапазона от 169.254.0.1 до 169.254.255.254.
