Rejstřík
802.11b nebo 802.11g
Sítě 802.11b pracují s maximální rychlostí 11 megabitů za sekundu. Sítě 802.11g pracují s rychlostí až 54 megabitů za sekundu.

Přístupový bod
Známý také jako bezdrátový směrovač. Přístupový bod je srdcem vaší bezdrátové sítě a řídí konfiguraci jejího zabezpečení. Pokud používáte přístupový bod, máte síť s režimem Infrastruktura.

Ad-hoc
Každé zařízení komunikuje přímo s každým dalším zařízením. Neexistuje žádný centrální přístupový bod, který by komunikaci v bezdrátové síti řídil. Tento typ sítě se označuje také jako peer-to-peer.

APIPA
Technologie APIPA (Automatic Private IP Addressing) se používá, když vaše síťové zařízení nemůže získat IP adresu ze serveru DHCP.

ASCII
American Standard Code for Information Interchange (ASCII). Číselné vyjádření znaku. Kód ASCII například pro písmeno A je 65.

Způsob ověřování
Typ bezdrátového ověřování používaný ve vaší síti.

Kanál
Bezdrátové sítě používají kanály. Každý kanál využívá jinou rádiovou frekvenci v přednastaveném pásmu. Všechna bezdrátová zařízení v bezdrátové síti budou používat stejný kanál, pokud tato zařízení nejsou připojena k jiným bezdrátovým přístupovým bodům, které se chovají jako brány do bezdrátové sítě.

DHCP
Server DHCP (Dynamic Host Configuration Protocol) přiděluje síťovým zařízením IP adresy.

DSL
Digital Subscriber Line (DSL), označuje se také jako ADSL nebo xDSL. Vysokorychlostní internetové připojení.

EAP
Protokol EAP (Extensible Authentication Protocol) poskytuje ověřovací rámec pro bezdrátové a pevné sítě. Protokol EAP využívá server RADIUS, který ověřuje uživatele v podnikových sítích, než jim je povolen přístup k bezdrátové nebo pevné síti.

Síťový kabel (ethernetový kabel)
Kabel, který se používá pro připojení vašeho zařízení k vaší pevné síti. Ethernetové kabely používají konektory RJ45 a kabel samotný se také označuje jako přímý.

Adresa brány
Adresa brány umožňuje zařízení komunikovat s jinými zařízeními v jiných sítích. Obecně lze říci, že adresu brány nemusíte konfigurovat.

HEX
Zkratka pro HEXADECIMÁLNÍ, číselný systém, který využívá číslice 0–9 a písmena A–F.

Infrastruktura
Každé zařízení komunikuje přes centrální přístupový bod. Toto je nejčastěji používaný typ bezdrátové sítě.

IP adresa
Každé zařízení v síti musí používat jedinečnou adresu. Taková adresa se nazývá IP adresa a je třeba ji zadat ručně nebo ji lze pomocí serveru DHCP přidělit automaticky. IP adresa je povinná v síti TCP/IP.

LEAP
Protokol LEAP (Lightweight Extensible Authentication Protocol) je protokol vyvinutý společností Cisco, který se používá pro ověřování 802.1X v bezdrátových sítích LAN. Podobně jako protokol EAP využívá i protokol LEAP server RADIUS, který ověřuje uživatele v podnikových sítích, než jim je povolen přístup k bezdrátové nebo pevné síti.

MAC adresa
MAC adresa (Media Access Control) je jedinečné 12místné číslo přidělené každému bezdrátovému zařízení.

Název uzlu
Jedinečný název bezdrátového zařízení. Obecně lze říci, že název uzlu vychází z MAC adresy.

Síla signálu
Síla bezdrátového signálu mezi vaším zařízením a přístupovým bodem, s nímž komunikujete.

Maska podsítě
IP adresa vyžaduje také masku podsítě. Maska podsítě se zadává ručně, nebo pokud používáte server DHCP, přiděluje se automaticky. Maska podsítě je povinná v síti TCP/IP.

SSID
Identifikátor SSID (Service Set Identifier ), označovaný také jako název sítě, je název, který používá vaše bezdrátová síť. V některých bezdrátových sítích není název SSID zveřejňován, aby se zvýšilo zabezpečení.

Přepínač
Přepínač umožňuje komunikaci mezi několika síťovými zařízeními.

TKIP
TKIP klíč (Temporal Key Integrity Protocol) eliminuje možnosti, jejichž prostřednictvím můžou narušitelé zneužívat slabá místa vznikající při používání ručně přidělovaných WEP klíčů. TKIP klíče se generují automaticky a jsou delší než WEP klíče, a v důsledku toho jsou bezpečnější než standardní WEP klíče.

WEP klíč
WEP klíč (Wireless Equivalent Protocol) je základní prvek zabezpečení bezdrátových sítí, který chrání vaši bezdrátovou síť před náhodným odposloucháváním. WEP klíč se často nazývá také síťový klíč.
