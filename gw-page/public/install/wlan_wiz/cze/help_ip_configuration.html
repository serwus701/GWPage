Konfigurace IP adresy
Bezdrátovou zařízení je třeba konfigurovat pro používání IP adresy. Existují dvě možnosti:

Získat IP adresu automaticky. Tuto možnost použijte, pokud vaše síť používá server DHCP.

Nastavit statickou IP adresu. Tuto možnost použijte, pokud vaše síť nepoužívá server DHCP a chcete IP adresu a nastavení masky podsítě zadat ručně.

Pokud si nejste jisti, kterou možnost máte zvolit, poraďte se se svým správcem systému.

Upozornění: Pokud zvolíte možnost Získat IP adresu automaticky, pokusí se zařízení získat IP adresu od serveru DHCP. Pokud nebude moci komunikovat se serverem DHCP, protože je server DHCP nedostupný, nebo protože jste nesprávně konfigurovali nastavení bezdrátové sítě, přejde zařízení na IP adresu typu APIPA v rozsahu 169.254.0.1 až 169.254.255.254.
