Konfiguracija IP adrese
Bežični uređaj mora biti konfiguriran za korištenje IP adrese. Postoje dvije moguće opcije:

Automatsko dobivanje IP adrese. Koristite ovu opciju ako mreža koristi DHCP poslužitelj.

Postavi statičku IP adresu. Koristite ovu opciju ako mreža ne koristi DHCP poslužitelj i želite ručno unijeti IP adresu i postavke maske podmreže.

Ako niste sigurni koju opciju odabrati, molimo da se obratite administratoru sustava.

Imajte na umu: Ako odaberete opciju „Automatski dobij IP adresu”, vaš uređaj će pokušati dohvatiti IP adresu iz DHCP poslužitelja. Ako nije u mogućnosti komunicirati s DHCP poslužiteljem jer DHCP poslužitelj nije dostupan ili ste netočno konfigurirali sigurnosne postavke bežične veze, uređaj se prebacuje na APIPA na osnovu IP adrese u rasponu od 169.254.0.1 i 169.254.255.254.
