Potvrda postavki bežične mreže
Za potvrdu svih postavki bežične veze kliknite na gumb Dalje.

Ako ste zadovoljni sa sigurnosnim postavkama bežične veze na svom uređaju no želite promijeniti IP adresu kliknite na gumb „Promijeni IP adresu”, ovo će vam omogućiti promjenu postavki IP adrese uređaja bez izmjene sigurnosnih opcija bežične veze.
