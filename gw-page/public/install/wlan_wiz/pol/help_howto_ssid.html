Gdzie można znaleźć nazwę SSID?
Informacje o identyfikatorze SSID można znaleźć w dokumentacji dostarczonej przez producenta bezprzewodowego punktu dostępu/routera.

Identyfikator SSID można znaleźć na plombie bezprzewodowego punktu dostępu/routera.



Jeśli nie masz pewności co do tych informacji, skonsultuj się z administratorem systemu, dostawcą usług internetowych lub producentem bezprzewodowego punktu dostępu/routera.
