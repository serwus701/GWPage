Konfiguracja adresu IP
Urządzenie bezprzewodowe musi zostać skonfigurowana/skonfigurowane tak, aby używało adresu IP. Istnieją dwie opcje konfiguracji:

Pobierz adres IP automatycznie. Użyj tej opcji, jeśli w Twojej sieci znajduje się serwer DHCP.

Ustaw statyczny adres IP. Użyj tej opcji, jeśli w Twojej sieci nie ma serwera DHCP i chcesz ręcznie wprowadzić ustawienia adresu IP i maski podsieci.

Jeśli nie masz pewności, którą opcję należy wybrać, skontaktuj się ze swoim administratorem systemu.

Uwaga: Jeśli wybierzesz opcję Pobierz adres IP automatycznie, urządzenie będzie próbować uzyskać adres IP z serwera DHCP. Jeśli nie możesz skontaktować się z serwerem DHCP, ponieważ jest on wyłączony lub ponieważ nieprawidłowo skonfigurowano ustawienia bezpieczeństwa bezprzewodowego, urządzenie przełączy się na korzystanie z adresu IP w trybie APIPA, w zakresie 169.254.0.1 do 169.254.255.254.
