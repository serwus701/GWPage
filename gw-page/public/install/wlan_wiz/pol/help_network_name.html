Nazwa sieci bezprzewodowej
Wprowadź nazwę sieci (SSID), z którą zostanie skojarzone urządzenie bezprzewodowe.

Jeśli urządzenie kontaktuje się z siecią w trybie Ad-hoc i w sieci nie występuje punkt dostępu, zaznacz opcję To jest sieć Ad-hoc bez punktu dostępu. Po wybraniu tej opcji należy również ręcznie wybrać używany kanał sieci. Jeśli nie wiesz, którego kanału sieci używasz, zapoznaj się z dokumentacją dostarczoną z komputerem i poszukaj informacji o ustawieniach kanałów.
