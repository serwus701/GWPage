Configurarea certificatului
Certificat client
Selectaţi certificatul clientului pentru autentificarea EAP-TLS. Vă rugăm să ţineţi cont de faptul că trebuie mai întâi să configuraţi certificatul în pagina "Configurare certificat" de pe pagina web internă (EWS) a dispozitivului dumneavoastră. (Dacă utilizaţi o altă metodă de autentificare cu excepţia EAP-TLS, nu trebuie să selectaţi certificatul clientului.)
Verificare certificat server
Selectaţi metoda de verificare a certificatului serverului. (Dacă utilizaţi LEAP, atunci nu mai trebuie să selectaţi metoda de verificare.)
  • Fără verificare
    Acest certificat este sigur şi nu necesită verificare.

  • Cert. CA
    Certificatul serverului emis de CA este verificat de certificatul CA instalat pe acest dispozitiv. Vă rugăm să ţineţi cont de faptul că trebuie mai întâi să configuraţi certificatul în pagina "Configurare certificat" de pe pagina web internă (EWS) a dispozitivului dumneavoastră.

  • Cert. CA + ID server
    Sunt verificate următoarele: certificatul serverului (emis de CA utilizând certificatul CA instalat pe această maşină) şi numele serverului (ID server). Vă rugăm să ţineţi cont de faptul că trebuie mai întâi să configuraţi certificatul în pagina "Configurare certificat" de pe pagina web internă (EWS) a dispozitivului.

ID server
Introduceţi codul de identificare al serverului, ID server, pentru verificarea certificatului serverului. ID-ul serverului este numele certificatului serverului.

Configurare certificat
Dacă utilizaţi un certificat client sau un certificat CA, trebuie mai întâi să configuraţi certificatul în pagina "Configurare certificat" de pe pagina web internă (EWS) a dispozitivului
