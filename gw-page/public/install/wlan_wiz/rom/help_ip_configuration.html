Configurare adresă IP
Dispozitivul wireless trebuie să fie configurat pentru a utiliza o adresă IP. Există două opţiuni posibile:

Obţinere automată adresă IP. Utilizaţi această opţiune dacă reţeaua dumneavoastră utilizează un server DHCP.

Setare adresă IP statică. Utilizaţi această opţiune dacă reţeaua dumneavoastră nu utilizează un server DHCP şi doriţi introducerea manuală a adresei IP şi a adresei măştii de subreţea.

Dacă nu sunteţi sigur în privinţa opţiunii care trebuie selectată, luaţi legătura cu administratorul dumneavoastră de sistem.

Reţineţi: dacă selectaţi opţiunea „Obţinere automată adresă IP”, dispozitivul dumneavoastră va încerca să obţină o adresă IP de la un server DHCP. Dacă nu puteţi comunica cu serverul DHCP deoarece acesta este indisponibil sau aţi configurat incorect setările de securitate wireless, dispozitivul va comuta la o adresă IP bazată pe APIPA în intervalul 169.254.0.1 – 169.254.255.254.
