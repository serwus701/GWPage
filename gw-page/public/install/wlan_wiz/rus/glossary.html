Глоссарий
802.11b или 802.11g
Максимальная скорость работы в сетях стандарта 802.11b составляет 11 Мбит/с. Скорость работы в сетях стандарта 802.11g составляет до 54 Мбит/с.

Точка доступа
Точка доступа, также известная как беспроводный маршрутизатор, образует ядро беспроводной сети и контролирует параметры ее безопасности. При использовании точки доступа сеть имеет режим инфраструктуры.

Прямое подключение (Ad-hoc)
Каждое устройство связывается непосредственно с другим устройством. Центральная точка доступа, контролирующая связь в беспроводной сети, отсутствует. Такой тип сети также называется одноранговой беспроводной сетью.

APIPA
Автоматическое назначение частных IP-адресов (APIPA) используется в том случае, если сетевое устройство не может получить IP-адрес на сервере DHCP.

ASCII
Американский стандартный код для обмена информацией (ASCII). Цифровое представление символа. Например, код ASCII буквы A - 65.

Метод проверки подлинности
Тип проверки подлинности в беспроводных сетях, использующийся в данной сети.

Канал
В беспроводных сетях используются каналы. Каждый канал использует свою радиочастоту в предварительно определенном диапазоне. В случае, если беспроводные устройства не подключены к другим беспроводным точкам доступа, которые действуют как шлюз беспроводной сети, все эти устройства в пределах беспроводной сети будут использовать тот же канал.

DHCP
Сервер протокола динамической настройки узлов (DHCP) назначает IP-адреса для сетевых устройств.

DSL
Цифровая абонентская линия (DSL), которая также называется ADSL или xDSL. Высокоскоростное подключение к Интернет.

EAP
Протокол Extensible Authentication Protocol (EAP), который обеспечивает механизм проверки подлинности для беспроводных и кабельных сетей. EAP использует сервер RADIUS, который выполняет проверку подлинности пользователей в корпоративных сетях перед тем, как им будет предоставлен доступ к беспроводной или проводной сети.

Сетевой кабель (кабель Ethernet).
Кабель, использующийся для подключения устройства к проводной сети. Для кабеля Ethernet используются разъемы RJ45. Этот кабель также называется кабелем с прямыми соединениями контактов.

Адрес шлюза.
Адрес шлюза позволяет устройству связываться с другими устройствами в других сетях. В общем настраивать адрес шлюза не требуется.

16-РИЧ.
Сокращение для "ШЕСТНАДЦАТЕРИЧНЫЙ". Это система нумерации, использующая цифры от 0 до 9 и буквы от A до F.

Инфраструктура
Каждое устройство связывается через центральную точку доступа. Это наиболее распространенный тип беспроводной сети.

IP-адрес.
Каждое устройство в сети должно использовать уникальный адрес. Этот адрес известен как IP-адрес. Он должен либо вводиться вручную, либо присваиваться автоматически сервером DHCP. В сети TCP/IP IP-адрес является обязательным.

LEAP
Протокол Lightweight Extensible Authentication Protocol (LEAP) - это протокол Cisco, использующийся для проверки подлинности в беспроводных локальных сетях стандарта 802.1X. Как и EAP, LEAP использует сервер RADIUS, который выполняет проверку подлинности пользователей в корпоративных сетях перед тем, как им будет предоставлен доступ к беспроводной или проводной сети.

MAC-адрес
Адрес управления доступом к среде (MAC) – это уникальный адрес из 12 цифр, который присваивается каждому сетевому устройству.

Имя узла
Уникальное имя для беспроводного устройства. В общем случае имя узла основывается на его MAC-адресе.

Мощность сигнала
Мощность беспроводного радиосигнала между устройством и использующейся точкой доступа.

Маска подсети.
Для IP-адреса также требуется маска подсети. Маска подсети вводится вручную или присваивается автоматически сервером DHCP. В сети TCP/IP маска подсети является обязательной.

SSID
Идентификатор Service Set Identifier (SSID), который также называется сетевым именем, - это имя, используемое беспроводной сетью. В некоторых беспроводных сетях имя SSID не объявляется с целью повышения безопасности.

Коммутатор
Коммутатор позволяет нескольким сетевым устройствам связываться друг с другом.

TKIP
Протокол TKIP (Temporal Key Integrity Protocol) позволяет избежать предсказуемости, которая может применяться злоумышленниками для нахождения слабых мест системы безопасности, появляющихся в результате присвоенных вручную ключей WEP. Ключи TKIP генерируются автоматически. Они более длинные, чем ключи WEP, и поэтому предлагают более высокий уровень безопасности по сравнению со стандартными ключами WEP.

Ключ WEP
Ключ протокола Wireless Equivalent Protocol (WEP) – это основная характеристика безопасности, которая защищает беспроводную сеть от случайного перехвата. Ключ WEP также называется ключом сети.
