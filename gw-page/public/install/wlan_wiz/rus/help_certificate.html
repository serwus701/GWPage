Настройка сертификата.
Сертификат клиента
Выберите сертификат клиента для проверки подлинности EAP-TLS. Обратите внимание, что сначала необходимо настроить сертификат на странице "Настройка сертификата" внутренней веб-страницы (EWS) устройства. (выбирать сертификат клиента при использовании способа проверки подлинности требуется только для EAP-TLS).
Проверка сертификата сервера
Выберите способ проверки сертификата сервера (при использовании LEAP способ проверки выбирать не нужно).
  • Без проверки
    Сертификат сервера надежен и не требует проверки.

  • Серт. ЦС
    Сертификат сервера, выданный ЦС, проверяется с помощью сертификата ЦС, установленного на этом устройстве. Обратите внимание, что сначала необходимо настроить сертификат на странице "Настройка сертификата" внутренней веб-страницы (EWS) устройства.

  • Серт. ЦС + идентификатор сервера
    Сертификат сервера (выданный ЦС с помощью сертификата ЦС, установленного на этом устройстве) и общее имя (идентификатор сервера) должны быть подтверждены. Обратите внимание, что сначала необходимо настроить сертификат на странице "Настройка сертификата" внутренней веб-страницы (EWS) устройства.

Идентификатор сервера
Введите идентификатор сервера для проверки сертификата сервера. Идентификатор сервера является общим именем сертификата сервера.

Настройка сертификата
Если используется сертификат клиента или сертификат ЦС, необходимо настроить сертификат на странице "Настройка сертификата" внутренней веб-страницы (EWS) устройства.
