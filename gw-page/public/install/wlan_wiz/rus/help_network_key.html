Настройка ключа сети
Введите соответствующий ключ сети и нажмите кнопку "Далее". Для каждого метода безопасности имеются свои правила.
Open System/Shared Key с WEP
Этот ключ представляет собой 64-битное или 128-битное значение, которое должно вводиться в формате ASCII или в шестнадцатеричном формате.
  • ASCII 64 (40) бит:
    Состоит из 5 букв, например: “WSLAN” (регистр учитывается).
  • Шестнадцатеричный 64 (40) бит:
    Состоит из 10 знаков шестнадцатеричных данных, например: “71f2234aba”.
  • ASCII 128 (104) бит:
    Состоит из 13 букв, например: “Wirelesscomms” (регистр учитывается).
  • Шестнадцатеричный 128 (104) бит:
    Состоит из 26 знаков шестнадцатеричных данных, например: “71f2234ab56cd709e5412aa2ba”.
WPA-PSK/WPA2-PSK и TKIP или AES
Используется предварительный ключ (PSK) длиной 8 или более знаков (макс. длина: 63 знака).
