Имя беспроводной сети
Введите имя сети (SSID), с которой будет связано устройство.

Если устройство связано с сетью с прямым подключением (Ad-hoc), которая не имеет точки доступа, установите параметр “Это сеть с прямым подключением (Ad-hoc), в ней отсутствует точка доступа”. Если установить этот параметр, необходимо также вручную выбрать используемый канал. Если вы не знаете, какой сетевой канал используется, обратитесь к документации, прилагаемой к компьютеру, и ознакомьтесь с информацией о настройке каналов.
