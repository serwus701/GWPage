Konfiguracija IP naslova
Brezžična naprava mora biti konfigurirana tako, da uporablja IP naslov. Na voljo sta dve možnosti:

Samodejno pridobi IP naslov. To možnost izberite, če vaše omrežje uporablja strežnik DHCP.

Nastavi statičen IP naslov. To možnost izberite, če vaše omrežje ne uporablja strežnika DHCP in želite ročno vnesti IP naslov in nastavitve podomrežne maske.

Če ne veste, katero možnost izbrati, se obrnite na skrbnika sistema.

Ne pozabite: če izberete možnost "Samodejno pridobi IP naslov", bo vaša naprava poskušala pridobiti IP naslov s strežnika DHCP. Če komunikacija s strežnikom DHCP ni mogoča, ker strežnik DHCP ni na voljo ali ste napačno konfigurirali brezžične varnostne nastavitve, bo naprava preklopila na APIPA glede na IP naslov v razponu od 169.254.0.1 do 169.254.255.254.
