Potrditev nastavitev brezžičnega omrežja
Za potrditev vseh brezžičnih nastavitev kliknite gumb Naprej.

Če ste zadovoljni z brezžičnimi varnostnimi nastavitvami na vaši napravi, a želite spremeniti IP naslov, kliknite gumb "Spremeni IP naslov" in tako boste lahko spremenili nastavitve IP naslova naprave brez spreminjanja brezžičnih varnostnih možnosti.
