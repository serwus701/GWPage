Konfigurácia adresy IP
Bezdrôtové zariadenie musí byť nakonfigurované tak, aby používalo adresu IP. K dispozícii sú dve možnosti:

Získať adresu IP automaticky. Túto možnosť použite v prípade, že vaša sieť používa server DHCP.

Nastaviť statickú adresu IP. Túto možnosť použite v prípade, že vaša sieť nepoužíva server DHCP a chcete ručne zadať nastavenia adresy IP a masky podsiete.

Ak si nie ste istí, ktorú možnosť vybrať, obráťte sa na správcu systému.

Upozornenie: Ak vyberiete možnosť Získať adresu IP automaticky, zariadenie sa pokúsi získať adresu IP zo servera DHCP. Ak zariadenie nedokáže komunikovať so serverom DHCP, pretože server DHCP je nedostupný alebo ste nesprávne nakonfigurovali nastavenia zabezpečenia bezdrôtovej siete, zariadenie sa prepne na adresu IP na základe režimu APIPA v rozsahu 169.254.0.1 až 169.254.255.254.
